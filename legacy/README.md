# Legacy Python / CustomTkinter app (v1)

This folder contains the previous Windows-focused Retail Invoice desktop app.

The current product is the Tauri + React app at the repository root (v2). Use **Setup → Import previous Retail Invoice database** to bring `data/invoices.db` into the new app.

To run the legacy app:

```bash
cd legacy
pip install -r requirements.txt
python main.py
```
