## Retail Invoice 1.7.0-beta

### What’s new

- **Product categories** (Cell Phones, Laptops, Desktops, Monitors, Printers, Accessories, Other) — add more in Setup
- **Download inventory** spreadsheet with Status = In inventory; Import only adds new SKUs
- Spreadsheet columns include **Category** and **Status**
- **Sales History** renamed to **Reports**, with **Print inventory** list

### Hotkeys

F1 New · F2 Products · F3 Reports · F4 Setup · F5/F6 product save · F10 Save · F11 Preview · F12 Complete + Print · F7/F8 payment · F12 confirm

### Requirements

- Windows 10 or 11
